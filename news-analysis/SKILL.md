# News Analysis Skill

将每日热点新闻搜索并整理写入 Obsidian。

## 工作流程

### 第一步：获取 Obsidian 路径

使用 `/obsidian` skill 获取本机 Obsidian vault 路径。

### 第二步：搜索热点新闻

使用 `mmx search query` 分多角度搜索今日热点新闻：

```bash
DATE=$(date +%Y-%m-%d)
mmx search query "国际新闻,国内时政新闻,财经新闻,科技AI新闻 $DATE"
```

### 第三步：整理新闻内容

对每条热点新闻：
1. 写 100 字以内的核心内容摘要
2. 保留原文链接
3. 标注新闻来源和时间

### 第四步：写入 Obsidian

1. 确定写入路径：`{Obsidian路径}/news/`
2. 文件命名格式：`$(date +%Y-%m-%d_%H:%M:%S)_{热点新闻}.md`
3. 按以下格式写入：

```markdown
# 今日热点新闻30条 | $(date +%Y-%m-%d)

## 🌏 国际动态
### [序号]. [新闻标题]
**摘要**：[100字以内核心内容]
**来源**：[媒体名称1](原文链接1) | [媒体名称2](原文链接2) | ...

---

...

```

## 注意事项
- 摘要需精炼，控制在 100 字左右
- 所有链接必须保留原文地址
- 趋势归纳部分要有深度分析，不只是罗列
- 文件名中时间精确到秒，避免重复

## 错误处理

- 若 Obsidian vault 不存在，提示用户检查路径
- 若 news 文件夹不存在，自动创建：`mkdir -p "{Obsidian路径}/news"`
- 若 mmx search 无结果，输出"今日未找到热点新闻"并退出
